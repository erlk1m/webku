# Panduan Astro untuk Pemula

Astro adalah framework web all-in-one untuk membangun website yang cepat.

## Apa itu Astro?

Astro memungkinkan kamu membangun website dengan:

- **Zero JavaScript by default** — tidak ada JS yang dikirim ke browser
- **Island Architecture** — hydrate hanya komponen yang interaktif
- **Support semua framework** — React, Vue, Svelte, dll

## Cara Install

```bash
npm create astro@latest

````
