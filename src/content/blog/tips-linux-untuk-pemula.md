# Tips Linux untuk Pemula

Berikut adalah beberapa tips yang berguna untuk kamu yang baru mulai menggunakan Linux.

## 1. Kenali Terminal

Terminal adalah senjata utama di Linux. Beberapa perintah dasar:

```bash
ls - list file
cd - ganti direktori
cp - copy file
mv - pindah/renama file
rm - hapus file

````
